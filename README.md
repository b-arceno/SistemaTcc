# User Management System

Este é um sistema de gerenciamento de pedidos desenvolvido com Node.js, Express e MySQL. Ele permite a criação, leitura, atualização e exclusão (CRUD) de produtospara que o usuário realize determinados pedidos, com uma interface web para administradores e usuários.

## Funcionalidades

- Cadastro e login de usuários
- Criação, edição e exclusão de usuários/produtos
- Diferenciação entre administradores e usuários comuns

## Tecnologias Utilizadas

- **Node.js**: Ambiente de execução para JavaScript no lado do servidor.
- **Express**: Framework para construção de aplicativos web com Node.js.
- **MySQL**: Sistema de gerenciamento de banco de dados relacional.
- **EJS**: Motor de visualização para renderizar páginas HTML.
- **Bootstrap**: Framework CSS para estilização das páginas.
